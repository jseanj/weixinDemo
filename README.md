weixinDemo
==========

利用微信sdk写的demo

- 此demo只实现了将app分享到朋友圈的功能
- 需要提供微信提供的appId以及分享消息的图片

记住：在Xcode中，选择你的工程设置项，选中“TARGETS”一栏，在“info”标签栏的“URL type“添加“URL scheme”为你所注册的应用程序id.
